JSON(JavaScript Object Notation)
: 데이터를 저장하고 교환하기 위한 경량 데이터 형식
➡ 키(Key)-값(Value) 쌍을 기반으로 데이터를 표현하며, 사람과 기계가 모두 읽기 쉬운 구조를 가짐.
➡ 웹 개발, API 데이터 교환, 데이터 저장 등에 널리 사용됨.

!! JavaScript와 Dictionary !!

✅ 가볍고 빠름 → 텍스트 기반의 단순한 구조
✅ 가독성 높음 → 사람이 읽고 쓰기 쉬움
✅ 언어 독립적 → 대부분의 프로그래밍 언어에서 사용 가능
✅ 데이터 교환 표준 → 웹 API(RESTful API)에서 많이 사용
