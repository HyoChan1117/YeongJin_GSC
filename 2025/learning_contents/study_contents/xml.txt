XML(eXtensible Markup Language) 정리
1. XML이란?
데이터를 저장하고 교환하기 위한 마크업 언어
문서의 구조나 스타일을 태그를 이용해 정의

2. XML의 특징
✅ 태그(Tag) 기반 → <태그명>데이터</태그명> 형태로 구성
✅ 중첩(Nested) 구조 → 계층적인 데이터 표현 가능
✅ 확장 가능 → 사용자가 원하는 태그를 자유롭게 정의 가능

3. 태그를 사용하는 이유
📌 데이터를 구조화하기 위해 사용
✔ 정보를 체계적으로 정리하여 쉽게 이해 가능
✔ 계층 구조를 활용해 데이터 관계 표현 가능
✔ 데이터 교환 시 의미를 명확하게 전달

태그 방식을 사용하기 때문에 파싱 하는데에 오랜 시간이 걸림
